# 測試計畫 — 2026-04-24（Riding 背景播放 + 鎖屏 Media Session 控制）

本計畫對應 `docs/plans/2026-04-24-riding-background-playback.md`，涵蓋三項核心改動：
1. **靜音 MP3 填充**：將 `playLoop` 內的 `await wait(N)` 全部改成 `await playMp3('audio/_silence/sil-NN.mp3')`，使音訊元素永遠在播放，iOS 不再 throttle JS 計時器。
2. **`playMp3` 改用 `addEventListener`**：每個 promise 綁自己的 `ended` listener（`{once:true}`），防止 skip race bug。
3. **Media Session API**：在 Riding Start 時於鎖屏 / Control Center / 藍牙耳機上顯示題號 + 四顆控制按鈕（⏮ ⏸/▶️ ⏭）。

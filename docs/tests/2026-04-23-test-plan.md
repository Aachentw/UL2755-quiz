# 測試計畫 — 2026-04-23（Star 書籤 + Riding 解析 + Back 按鈕整理補丁）

本計畫對應 `docs/plans/2026-04-23-star-and-riding-polish.md`，涵蓋三個功能變更：
1. **星標書籤**：MCQ 右上 ⭐、Question List 過濾 bar、Settings Clear stars、`SrsStore` 星標 API。
2. **Riding 解析文字**：答案階段顯示 `💡 explanation`，問題階段隱藏。

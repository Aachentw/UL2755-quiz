# 測試計畫 — 2026-04-27（v0.74 Riding 背景播放修復後驗證）

## 範圍

接續 2026-04-23 與 2026-04-24 兩份測試計畫。本份新增 **TC-S（Simulated iPhone）** 系列，目的是把原本只在 iPhone 實機才能驗的 TC-M 轉為可在桌面 Chromium 自動化的測試，提高自動化覆蓋率。

執行環境：桌面 Chromium（preview_eval driver）跑在 `http://localhost:51914/`，PWA 載入 `v0.74` 程式碼。`srs.js` 純邏輯測試另開 iframe 載入 `srs.test.html` 由瀏覽器內 assert 系統自動回報。
